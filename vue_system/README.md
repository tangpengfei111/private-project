# vue.js + node.js  后台管理平台

> 技术：vue2，axios，vue-router，Node.js，WebPack，EXpress

前端页面使用vue构建，后端服务器由Node.js的Express框架搭建，网络请求为axios。

## 功能
* 用户注册登录、退出登录
* 添加手机信息，实现用户、手机关联
* 用户分组、部门之间进行关联
* 用户操作查询


## 说明
> 分别进入vue_system_front_end和vue_system_server安装依赖、启动项目

``` bash
# 安装依赖
npm install

# 启动项目
npm run start
npm run serve

# 打包发布
npm run build

```
